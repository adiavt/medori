# Medori
A very simple custom type for working with Date, Datetime & Time especially when working with SQL datetime format

## Usage
`go get github.com/dwadp/medori`

## Accepted Date format
* `YYYY-MM-DD`
* `DD-MM-YYYY`
* `YYYY/MM/DD`
* `DD/MM/YYYY`

## Accepted Datetime format
* `YYYY-MM-DD HH:MM:SS`
* `DD-MM-YYYY HH:MM:SS`
* `YYYY/MM/DD HH:MM:SS`
* `DD/MM/YYYY HH:MM:SS`

## Time format
* `HH:MM:SS`
